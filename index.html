What is a Neural Network?

A neural network is a computer system modeled on the human brain and nervous system. The amazing thing about it is that it can learn by itself, just like a brain! It's design is perfect for identifying patterns and that is why it is used in image classification.

Try out Cleverbot

Cleverbotclose

Cleverbot is a web application that uses an AI algorithm to have conversations with humans. Unlike other chatterbots, Cleverbot's responses are not programmed. Instead, it "learns" from human input. Based on the turing test, Cleverbot is judged to be 59.3% human, compared to the rating of 63.3% human achieved by human participants.

Google voice search allows users to use Google Search by speaking on a mobile phone or computer, i.e. have the device search for data upon entering information on what to search into the device by speaking. This is achieved through an AI algorithim is able to recognize 30+ languages.




What I'm Doing

I am creating an image recognition web app to see the effects of good and bad data. The goal of this project is to find out how bad training data affects neural networks. And secondly, if can it recover from one or more bad input.
